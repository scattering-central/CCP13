3.1.4 Extraction parameters

Figure 1: Extraction parameters

These parameters apply to the extraction of structural information from the 1-D correlation function and also give the user the option to output the interface distribution function. The user can choose to calculate the interface distribution function by selecting the checkbox shown in Figure 1. This option must be selected prior to the Transform process. Other options must be set prior to the Extraction process.

The estimate of the volume fraction crystallinity is used in Porod analysis if the lamellar interpretation of the 1-D correlation function fails (see Extraction).

The user-control of extraction checkbox allows the user to manually select the linear region of the 1-D correlation function D0 to Dtr (see Extraction and Overview). This region is selected automatically by default, but the values of D0 and Dtr obtained should be carefully examined and manual selection performed if necessary. The manual selection of the range D0 to Dtr is performed by selecting the checkbox provided and then clicking on the Edit Limits button. The range D0 to Dtr for each frame of the 1-D correlation function can then be selected in the same way as the variable tail-fitting limits were selected in the Fitting parameters section.
