3.1.3 Transform Parameters

Figure 1: Transform parameters

These parameters are used in the Transform process, in which the 1-D correlation function and second moment of the data are calculated from the extrapolated data. The user must specify the maximum d-spacing and step in d for the correlation function, and can also choose whether to re-transform the 1-D correlation function to obtain a simulated scattering curve which can be compared to the original data.

To generate a volume fraction profile, select Transform type: Hilbert (see Hilbert).
