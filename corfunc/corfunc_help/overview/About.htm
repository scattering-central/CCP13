About CORFUNC

The set of routines that comprise CORFUNC were originally written in 1994 at the behest of Tony Ryan by Thomas (Tom) Nye whilst an undergraduate in Applied Mathematics at the University of Cambridge, and are based on the earlier work of Gert Strobl (Universität Freiburg) and co-workers in the 1980's (see References). The polydispersity analysis was proposed later by Guy Eeckhaut (ex-ICI).

The original UNIX source was ported to the PC/WindowsTM environment and given a JavaTM GUI by CCP13 Post-Doctoral workers Richard Denny and Mark Shotton in 1999.

The alternative Hilbert transform (volume fraction profile) analysis of Trevor Crowley (ex-Salford University) and co-workers (see References) was suggested by Steve King (ISIS Facility). Its incorporation was started by CCP13 Post-Doctoral worker Matthew Rodman and finally implemented in 2004 by Steve King and Damian Flannery (ISIS Facility), with help from Richard Heenan (ISIS Facility).

 

About JavaTM

CORFUNC requires JavaTM; please download the Java 2 Run-time Environment (J2RE) from http://java.sun.com/j2se/ (you do not require the Software Development Kit, SDK). CORFUNC Version 1.3 has been extensively tested with Java 2 Version 1.3.1. though at the time of writing the current release is Version 1.4.2. During limited testing under Version 1.4.2 a small compilation time error was discovered. This has been corrected by changing the CORFUNC Java code but should perhaps serve as an indication that other run time "peculiarities" may exist. If you think you find one, please notify the CCP13 Software Development Team (http://www.ccp13.ac.uk/committee/committee.htm).

 

Steve King, August 2004.
