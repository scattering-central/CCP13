4 Actions Menu

Figure 1: The Actions menu

By selecting items in the Actions menu, the user can perform any of the three stages involved in the analysis:
 
Extrapolate Extrapolates the scattering curve to q = 0 and q = infinity
Transform Calculates the appropriate transform of the extrapolated data
Extract Parameters Interprets the 1-D correlation function; parameters describing the volume fraction profile are extracted automatically

These processes, and selection of the type of analysis to be performed, use input parameters which can be edited by selecting the Parameters item from the Edit menu.
