6 Graphics Windows

Figure 1: The CORFUNC graphics window showing the display menu.
 

A CORFUNC graphics window (see Figure 1) is opened whenever the user requests data to be plotted (see the Display menu). It contains the following functionality:

 

6.1 Zoom-in
The user may zoom-in on any region of the graph by dragging out a rectangular area with the left-hand mouse cursor (see Figure 2). The user can zoom-out again by selecting the Fit to window item from the Display menu.

Figure 2: A zoomed-in view of the extrapolated data (magenta) superimposed on the original SAXS data (blue).

6.2 Move limits
In the case of scattering data display, the user may move the red limit markers (see Fitting parameters), by dragging the markers with the right-hand mouse button.

6.3 Show co-ordinates
The co-ordinates of the mouse position within the graphics window may be displayed by selecting the Coordinates item from the Display menu (see Figure 1). The co-ordinates will be displayed in a separate window as shown in Figure 3.

 
Figure 3: The co-ordinate display window.

6.4 Display multiple frames
The user may display multiple frames (when available) of the data in the graphics window by selecting the Data frames ... item from the Display menu (see Figure 1). Only the first frame of the data is shown by default. When extrapolated data (see Extrapolation) or re-transformed data (see Transform) are superimposed on the original scattering data, multiple frames may be displayed by selecting the Extrapolation frames ... and Re-transform frames ... items (see Figure 1).

 

Figure 4: The frame chooser window.

6.5 Fit to window
This item from the Display menu causes the display to zoom-out to the point where all data is displayed within the window.

6.6 Superimposed data
After performing an extrapolation or re-transformation of the 1-D correlation function, the extrapolated data (magenta) or re-transformed data (green) are shown superimposed on the original scattering data (blue) for comparison (see Figure 5).

Figure 5: The CORFUNC graphics window showing re-transformed data (green)  and extrapolated data (magenta) superimposed on the original SAXS data (blue).

6.7 Save as...
The contents of the graphics window can be saved in either TIFF, JPEG or bitmap format. The format will be chosen according to the file extension entered:

.tiff or .tif      TIFF format (default)
.jpg              JPEG format
.bmp            bitmap format
