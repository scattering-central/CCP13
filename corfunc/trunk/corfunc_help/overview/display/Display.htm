5 Display Menu

Figure 1: The Display menu
 

The Display menu contains a list of checkboxes that allow the user to plot the results of any analysis. If a checkbox is selected, the appropriate graph will be displayed once it becomes available. De-selecting a checkbox will close the graph if it is already displayed. Closing the graph windows will cause the corresponding checkboxes to be de-selected. Certain graphs are only calculated when analysing more than one frame of data, and others are only calculated at the user's request. For example, a volume fraction profile (checkbox "Hilbert" cannot be displayed if correlation function analysis has been performed).

Always calculated:
 
Data The original scattering data (see File)
Extrapolation The extrapolated data (see Extrapolation)

Calculated when analysing more than one frame of data:
 
Guinier radius Guinier radius of gyration v. frame (see Extrapolation)
Thermal background Bonart thermal backround v. frame (see Extrapolation)
Porod constant Porod constant v. frame (see Extrapolation)
Sigma Sigma v. frame (see Extrapolation)
Second moment Second moment of the data v. frame (see Transform)

Calculated at request of user:
 
Gamma1 The 1-D correlation function (see Transform)
Hilbert The Volume Fraction Profile (see Transform)
Re-transform The Fourier transform of the 1-D correlation function (see Transform)
Interface distribution The Interface Distribution Function (see Transform)

The extrapolated data and re-transform are superimposed on the original scattering data when displayed. All other graphs are displayed in separate windows. The use of CORFUNC graphics windows is explained in Section 6: Graphics Windows.
