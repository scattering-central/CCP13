Together with the fluctuations introduced by this truncation, fluctuations are also introduced into the correlation function by the finite gap between points in the extrapolated dataset, i.e. we don't have intensity as a continuous function of q.

The user must specify the maximum value of R to which the correlation function should be calculated, the step in R, and whether to re-transform the correlation function for comparison with the original scattering data. These parameters may be input in the Transform parameters section of the Parameters window, opened by choosing the Parameters... item from the Edit menu.

4.2.1.3 Tips

(1) Check that Γ1 tends to zero as q tends to infinity; otherwise the background subtraction (see Extrapolation) may be incorrect.

(2) The gradient of Γ1 at R = 0 should be zero.

(3) There should be no ripples corresponding to q = 0.6 (R = 10Å) or the joins in the Extrapolated data.

(4) Choose the maximum R and step in R to show up anticipated spacings in the sample.

4.2.1.4 Next Steps

Once the 1-D correlation function has been calculated successfully, the user may interpret this in order to obtain structural parameters describing the sample (see Extraction of parameters).

The extrapolated dataset is interpolated to provide data with an equally-spaced Q-axis for input to the Hilbert transform routine. The interpolation is performed out to Q = 10.0 but only data out to Q = 0.5 is actually transformed. Every point in the transform yields a point on the segment density distribution. 4.2.2.3 Next Steps
