4.3.3.2 Output

The following files are output during the transformation process (assuming scattering data filename A01000.nnn and q-axis file X00000.nnn where n is any alphanumeric character):

In all cases:
A01VFR.TXT Volume fraction profile (ASCII format)
TROPUS.TXT Text file containing the normalisation parameters, adsorbed layer thicknesses, and bound fraction (ASCII format)
