4.3 Extraction of Parameters
 

4.3.1 Interpretation of the 1-D Correlation Function

Once the 1-D correlation function has been calculated by Fourier transforming the extrapolated data, the user may interpret the correlation function by selecting the Extract parameters item from the Actions menu. The correlation function is interpreted in terms of an ideal lamellar morphology [7] and structural parameters are obtained as shown in Figure 1 below. It should be noted that a small beam size is assumed; no de-smearing is performed.


Figure 1: Interpretation of the 1-D correlation function Γ1(R) assuming an ideal lamellar morphology. The structural parameters obtained (see [1]) are :

Long period  = Lp
Average hard block thickness = Lc
Average core thickness = D0
Average interface thickness = Dtr
Average soft block thickness = La = Lp - Lc
Local crystallinity = φ1 = Lc / Lp
Bulk crystallinity = φ = Γmin / (Γmin + Γ*)
Polydispersity = Γmin / Γmax
Electron density contrast = (ρc - ρa)² = (Δρ)² = Q Γ*/ (φ (1 - φ))
Specific inner surface = Os = 2φ / Lc
Non-ideality = (Lp - Lp*)² / Lp²

where Q = Invariant (see Transform)

 

4.3.1.1 Important Considerations

(1) It is important to note that, according to Babinet's principle, in a two-phase structure such as the ideal lamellar morphology, the electron (or neutron scattering length) densities of the two phases (crystalline and amorphous) may be interchanged without affecting the scattering curve or correlation function [6]. Therefore, from correlation function analysis alone, we cannot distinguish between:

Lc and La
φ and (1 - φ)
ρc and ρa

and these parameters may be interchanged. This will affect the values of any parameters derived from them!

(2) The CORFUNC program will attempt to determine Dtr and DO (the start and end of the linear section of Γ1(R) (see Figure 1) automatically. However, if these values appear to be incorrect on inspection, the user should select the values of Dtr and Do manually for each frame in the dataset (see Extraction parameters, derived from the Parameters... item in the Edit menu). These parameters affect almost all other structural parameters that are extracted.

(3) It is also important to check that a horizontal section in the first minimum of the 1-D correlation function (-Γmin) exists, as shown in Figure 2. This horizontal region is usually present when the sample crystallinity is <30% or >70%. In the intermediate region between 30% and 70% crystallinity, there may be no horizontal region, and in this case, the equation for the bulk crystallinity:

Bulk crystallinity = φ = Γmin / (Γmin + Γ*)

no longer strictly applies, yielding a less accurate value for φ and affecting the other parameters derived from it (i.e. polydispersity, Δρ and Os). In order to obtain reliable results for this intermediate crystallinity, other data is needed to complement the correlation function analysis [6].

If a local minimum below the abscissa followed by a local maximum above the abscissa are not found, the interpretation based on the ideal lamellar morphology is abandoned for that frame of data.


Figure 2: The 1-D correlation function shown in blue has a horizontal section in the first minimum, yielding the bulk crystallinity from the base line -Γmin . The first minimum of the 1-D correlation function shown in magenta has no horizontal section and the equation for bulk crystallinity given above no longer strictly applies.
 

4.3.2 Porod Analysis

If a Porod profile was used for tail-fitting, Porod analysis is performed after extraction of the lamellar structure parameters. If the lamellar interpretation of Γ1 failed, the crystallinity estimate provided by the user (see Extraction parameters, derived from the Parameters... item in the Edit menu) is used in the analysis, otherwise the bulk crystallinity described in section 4.3.1 above is used. The parameters calculated by the Porod analysis are given below:
 
Porod constant K (see Extrapolation)
Surface to volume ratio  = π K φ (1 - φ) / Q
Characteristic chord length lp = 4Q / π K
Crystalline chord length  lp / (1 - φ)
Amorphous chord length  lp / φ

where Q = Invariant (see Transfrom)

4.3.4 Output

The parameters described in sections 4.3.1 and 4.3.2 are output to the Message window and are also saved in an external ASCII file named:

A01000.LIS

(assuming scattering data filename A01000.nnn where n is any alphanumeric character)

An ASCII text file EXTRACT.TXT is also written. This contains the parameters used in the extraction process.
