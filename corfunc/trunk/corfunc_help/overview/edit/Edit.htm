3 Edit Menu

Figure 1: The Edit menu
 

3.1 Edit Parameters ...

The input parameters to the extrapolation, transform and parameter extraction processes (see Actions) can be edited by selecting the Parameters ... item from the Edit menu (see Figure 1). Upon selection, the user is presented with a tabbed pane as shown in Figure 2. This menu item cannot be selected until scattering data has been loaded (see the File menu).

Figure 2: The tabbed pane containing the input parameters

Notice that the Transform and Hilbert tabs are mutually exclusive depending on the setting of Transform type:

 

For Correlation Function Analysis... Select Standard
To generate a Volume Fraction Profile... Select Hilbert

The parameters are arranged into four categories corresponding to different stages of the analysis. These are:

 

(3.1.1) General
(3.1.2) Fitting
(3.1.3) Transform
(3.1.4) Extraction

(3.1.5) Hilbert

New parameter values can be set by clicking on the Apply button. All changes made since opening the tabbed pane will be cancelled by either closing the window or clicking on the Cancel button.
 

3.2 Clear messages

Selecting this menu item will clear the Message Window.
 

3.3 Next Steps

Once the input parameters have been set correctly, the user may perform the next stage of the analysis (see Actions).
