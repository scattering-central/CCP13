3.1.1 General parameters

Figure 1: General parameters
 

These parameters relate to all stages of the analysis and must be set prior to the extrapolation process. The user must choose which frames of the input data file are to be considered in the analysis (all frames of the input file by default) and whether the data are in arbitrary or absolute intensity units. If the absolute data scale is chosen, the data units are assumed to be reciprocal centimetres.
